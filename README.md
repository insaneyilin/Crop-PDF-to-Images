# Crop-PDF-to-Images

"Crop" one PDF page to images.

A very simple script using Python and [OpenCV](http://opencv.org).

### USAGE: 

Put images in the "images" directory.

Run the command

```
python pdfcrop.py
```

Results will be in the "output" directory.
